# link-aggregator-apps

A link aggregator based on JavaScript and AppsScript.

It should:

- Have a link
- Have a title
- Have a summary of the link
- Be upvotable and have an upvote count

It should also support dark mode and a regular website

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/link-aggregator-apps.git
cd link-aggregator-apps
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
